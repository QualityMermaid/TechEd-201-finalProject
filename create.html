How to generate story prompts:

You can use the form below to add customization options for your story prompts. The story prompt will then select from the list you have given. If you want a certain name/game just add one.

Make sure to seperate your inputs with a ',' eg. Rachel, Jessica, Jenny, Sarah

When asked for a time like Bedtime please add one number between 0-11. eg. 8 for 8pm
